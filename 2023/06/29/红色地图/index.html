北京市红色景区地图系统

技术栈

  • Spring Boot
  • Vue3
  • MySQL
  • MyBatis
  • axios
  • nginx

开发环境配置

  • 安装IDEA, JDK18
  • 安装MySQL, 地址,配置环境变量
  • 创建数据库gaode, 导入数据gaode.sql。如已存在，先drop database gaode;
  • 安装vue3
  • 配置环境变量,JAVA_HOME, MAVEN_HOME, path

部署

  • 打包后端文件, mvn package
  • 安装前端依赖，进入前端目录，npm install
  • 启动后端，java -jar gaode-java.jar
  • 启动前端，npm run serve
  • 或在ecs云服务器中启动脚本./start-app.sh

nginx反向代理

修改/etc/nginx/nginx.conf. http中location,指定要代理的ip和端口号

note

启动程序时可能会遇到权限错误。
解决方法：

  • sudo chmod 777 -R folder (程序所在目录)
  • ./start-app.sh
